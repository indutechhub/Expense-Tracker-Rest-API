# Expense Tracker Rest API

This is a REST API built using Flask to manage expenses.

# Features

* User authentication (JWT)
* Add expenses
* Get expenses
* Update expenses
* Delete expenses


# Tech Stack

* Python
* Flask
* SQLite
* JWT

# How to Run

1. Install dependencies
2. Run app.py
3. Use Postman to test APIs

# API Endpoints

* POST /register
* POST /login
* POST /expenses
* GET /expenses
